# Setup Guides

Follow these guides **in order** to get the project running on your machine.

| # | Guide | What it sets up |
|---|-------|-----------------|
| 1 | [Git & GitHub](./01-github-setup.md) | Clone the repo, branching strategy |
| 2 | [Node.js & npm](./02-nodejs-setup.md) | Install Node, run frontend + backend |
| 3 | [Firebase Auth](./03-firebase-setup.md) | Create Firebase project, get API keys |
| 4 | [MongoDB Atlas](./04-mongodb-setup.md) | Create free cluster, get connection string |
| 5 | [Vultr Object Storage](./05-vultr-setup.md) | Set up image storage bucket |
| 6 | [Google Gemini API](./06-gemini-setup.md) | Get Gemini API key for AI chatbot |
| 7 | [ElevenLabs](./07-elevenlabs-setup.md) | Get TTS API key and voice ID |
| 8 | [CRS Credit API](./08-crs-setup.md) | Configure CRS sandbox for credit screening |
| 9 | [Tambo AI](./09-tambo-setup.md) | Get Tambo API key for chat UI |
| 10 | [Environment Variables](./10-env-variables.md) | Fill in all `.env` files |
| 11 | [Running the Project](./11-running-locally.md) | Start frontend + backend + seed data |

**Estimated total setup time: 30-45 minutes**

> Tip: Split the work! One person can do Firebase + MongoDB while another does Vultr + API keys.
